# ❤️ Beating Heart Animation

A visually engaging web project that simulates a realistic beating heart using smooth animations. Built with HTML, CSS, and JavaScript, this project focuses on creative UI design and animation effects to deliver an eye-catching experience.

---

## 🚀 Features

- ❤️ Realistic beating heart animation  
- 🎨 Smooth CSS/JS animation effects  
- ⚡ Lightweight and fast  
- 📱 Responsive design  
- 💻 Clean and minimal UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
